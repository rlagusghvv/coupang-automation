# Couplus MVP Roadmap

## 목표 (MVP)
- 도매꾹 상품 URL → 업로드/미리보기까지 **한 화면에서** 처리
- 서버는 **쿠팡 키 없이도 실행 가능** (미리보기/세션관리/UI 확인 목적)
- 로그인 세션 파일은 **레포 밖**(`~/.couplus`)에 저장
